WEEK 7
Week 7 Lab Manual: Programs using Recursion
Recursion is a technique where a function calls itself to solve a problem by breaking it into smaller subproblems. Every recursive solution must have:
      1. Base Case - stops recursion
      2. Recursive Case - reduces the problem size
________________


Pre-Lab 1:  Recursive Calculation of Factorial
Description
Write a recursive function to compute the factorial of a given non-negative integer.
Factorial of n is defined as:
      * n! = 1 if n = 0
      * n! = n × (n-1)! if n > 0
This problem introduces the simplest recursive structure.
Input Format
A single integer n (n ≥ 0)
Expected Output
Factorial of the given number
Example
Input:
5


Output:
Factorial = 120


________________

Fibonacci sequence is defined as:

      * F(n) = F(n-1) + F(n-2) for n > 1
This problem demonstrates multiple recursive calls.
Input Format
A single integer n (n ≥ 0)
Expected Output
The n-th Fibonacci number
Example
Input:
6


Output:
Fibonacci number = 8
